Este repositório é dedicado ao **aprofundamento em Python e IA**.

Aqui estão reunidos conteúdos, exemplos e práticas para evoluir em:

- Fundamentos e recursos avançados de Python
- Estruturas de dados e programação orientada a objetos
- Desenvolvimento de APIs e aplicações
- Conceitos e aplicações de Inteligência Artificial

O objetivo é servir como base de estudo contínuo, com foco em prática e evolução técnica.

## Executar no Google Colab

1. Clique no botão **Open in Colab**.
2. Execute as células do notebook `trilha_python_colab.ipynb` em ordem.
3. O notebook clona o repositório em `/content/trilha-python-dio`, instala dependências (se houver) e roda um exemplo com `UserProfile`.
